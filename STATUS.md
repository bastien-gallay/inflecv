# Project status

**Current posture:** `active` — solo maintainer, accepting issues and PRs
on a best-effort cadence.

| Posture | What it means |
|---|---|
| **active** | Issues triaged within ~1 week. PRs reviewed within ~2 weeks. |
| **slow** | Same as active, but expect 4-week turnaround. The project is alive but the maintainer is busy. |
| **maintenance** | Security fixes only. Feature PRs may be closed without review. Fork freely. |
| **archived** | No further maintenance. The repo is read-only / archived on GitHub. |

This file is the canonical signal. If GitHub Issues are stale and the
last commit is months old, check here before assuming the project is
abandoned — and check here before getting upset about a slow review.

Posture changes are announced in `CHANGELOG.md`.
